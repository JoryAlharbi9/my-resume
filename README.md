My Resume Web Page

📌 Description
This is my personal online resume website that I designed and developed using HTML and CSS.
The website presents my background as a Software Engineering student,
including my education, projects, certificates, and technical development tools, in a structured and visually organized way.

🎯 Features
- Fully responsive design using media queries (375px, 768px, 992px)
- Structured layout using semantic HTML (header, main, section, footer)
- Projects section with images, descriptions, and preview links (PDF)
- Education section with courses displayed as styled cards
- Certificates displayed in a responsive gallery layout
- Clean UI design using CSS (Flexbox, spacing, colors, and typography)

🛠️ Technologies Used
- HTML (for structure and content)
- CSS (for styling and layout)
- Flexbox & Media Queries (for responsiveness)
